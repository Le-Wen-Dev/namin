# Cách kích hoạt theme VMST Solutions

## Phương pháp 1: Qua trình duyệt (Khuyến nghị)

1. Mở trình duyệt và truy cập:
   ```
   http://localhost:your-port/activate-vmst-theme.php?activate=1
   ```
   (Thay `your-port` bằng port của Local Sites, ví dụ: `namin.local`)

2. Theme sẽ được kích hoạt tự động

3. **QUAN TRỌNG**: Xóa file `activate-vmst-theme.php` sau khi kích hoạt xong!

## Phương pháp 2: Qua WordPress Admin

1. Đăng nhập vào WordPress Admin
2. Vào **Appearance > Themes**
3. Tìm theme **"VMST Solutions"**
4. Click **"Activate"**

## Phương pháp 3: Qua WP-CLI (nếu có)

```bash
wp theme activate vmst-solutions
```

## Sau khi kích hoạt

1. ✅ Vào **Appearance > Menus** để tạo menu
2. ✅ Vào **Appearance > Customize** để tùy chỉnh
3. ✅ Xem website để kiểm tra theme mới
