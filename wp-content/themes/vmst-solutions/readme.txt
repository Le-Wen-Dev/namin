== Description ==

VMST Solutions is a beautiful, modern WordPress theme inspired by Squarespace's clean and professional design. This theme features:

* Responsive design that works on all devices
* Clean, modern layout
* Custom navigation menus
* Widget-ready footer
* Custom logo support
* Contact: 0832 57 59 05

== Installation ==

1. Upload the theme files to the `/wp-content/themes/vmst-solutions` directory
2. Activate the theme through the 'Themes' menu in WordPress
3. Configure your menus in Appearance > Menus
4. Customize your site in Appearance > Customize

== Frequently Asked Questions ==

= Does this theme support custom menus? =

Yes, the theme supports two menu locations: Primary Menu and Footer Menu.

= Can I customize the colors? =

Yes, you can customize colors through the WordPress Customizer.
